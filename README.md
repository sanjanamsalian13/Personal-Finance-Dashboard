# 💰 Personal Finance Dashboard

## Overview

An interactive web application built using Python and Streamlit that helps users analyze income, expenses, and savings through interactive visualizations.

---

## Features

- Upload CSV transaction files
- Automatic financial summary
- Filter transactions
- Search transactions
- Download filtered data
- Expense distribution pie chart
- Category-wise bar chart
- Daily spending trend
- Monthly expense summary

---

## Tech Stack

- Python
- Pandas
- Plotly
- Streamlit
